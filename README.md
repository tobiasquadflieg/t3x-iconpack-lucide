# Feather Icons for TYPO3

TYPO3 CMS Extension "iconpack_lucide"


## What does it do?

This Extension registers an Iconpack-Provider for EXT:iconpack which allows you to use the [Lucide Icons](https://lucide.dev/) in TYPO3.


## Installation/Configuration

1. Install EXT:iconpack from TER/git or with composer
2. Install this Extension from TER/git or with composer
3. Customize configuration (Optional)
